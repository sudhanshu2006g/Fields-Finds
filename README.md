# Fields-Finds- Agricultural Tools Rental System

PROBLEM STATEMENT​
Farming tools are expensive and not affordable for many small
farmers​
Many tools remain unused when not in regular use​
Farmers struggle to find tools when needed​
No proper digital platform for tool lending and rental

KEY FEATURES
Tool listing with admin approval​
Pickup & return proof upload​
Ratings and reviews after booking completion​
Dispute reporting​
In-platform chat (borrower ↔ lender)​
Weather information for farming planning

TECHNOLOGIES USED
Frontend​-HTML5, CSS3, Bootstrap 5, JavaScript​
Backend​-Django with Python​
Database​-SQLite ​
Architecture-Client–Server
