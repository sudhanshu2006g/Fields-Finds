# Fields-Finds- Agricultural Tools Rental System

**PROBLEM STATEMENT**-->
​
**1**.Farming tools are expensive and not affordable for many small farmers
​
**2**.Many tools remain unused when not in regular use
​
**3**.Farmers struggle to find tools when needed
​
**4**.No proper digital platform for tool lending and rental

**TECHNOLOGIES USED**

**1**.Frontend​-HTML5, CSS3, Bootstrap 5, JavaScript
​
**2**.Backend​-Django with Python
​
**3**.Database​-SQLite 
​
**4**.Architecture-Client–Server
